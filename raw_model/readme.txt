В данной папке находятся json файлы с текстовым описанием api-сущностей.
json-файлы для каждой api-сущности формируются на этапе анализа сервиса. Далее по описанию для каждой сущности формируется pydantic-модель
В описании для каждой api-сущности необходимо указывать все поля (для body), их типы, ограничения и тд.
